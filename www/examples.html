The following Fortran 95 example programs can be found in the SHTOOLS examples subdirectory
SHCilmPlus This program demonstrates how to expand spherical harmonic files into gridded maps using the GLQ routines, and how to compute the gravity field resulting from finite amplitude surface relief. The effects of aliasing are given special attention.
SHExpandDH This program demonstrates how to expand a grid that is equally sampled in latitude and longitude into spherical harmonics using the sampling theorem of Driscoll and Healy (1994).
SHExpandLSQ This program demonstrates how to expand a set of irregularly sampled data points in latitude and longitude into spherical harmonics by use of a least squares inversion.
SHMag This program demonstrates how to expand scalar magnetic potential spherical harmonic coefficients into their three vector components and total field.
MarsCrustalThickness This program computes a crustal thickness map of Mars given a variety of user inputs.
SHRotate This program demonstrates how to determine the spherical harmonic coefficients for a body that is rotated with respect to its initial configuration.
SHLocalizedAdmitCorr This program demonstrates how to calculate localized admittance and correlation spectra for a given set of gravity and topography spherical harmonic coefficients.
TimingAccuracy These programs calculate the time required to perform the GLQ and DH spherical harmonic transforms and reconstructions and the accuracy of these operations.
