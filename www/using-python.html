Using SHTOOLS with Python

To access the SHTOOLS routines from Python, it is necessary that the main SHTOOLS directory be in the Python search path. This can be accomplished by using the command

setenv PYTHONPATH = "shtoolslocation"

in a C-shell, or by using the command

export PYTHONPATH="shtoolslocation"

in a bash shell, where "shtoolslocation" is the root directory of the SHTOOLS directory, typically /usr/local/shtools. After entering the Python environment, it is only necessary to execute the statement

import pyshtools as shtools

to load the routines and documentation. If you are using iPython, which adds improved functionality to Python and which is highly recommended, the available shtools routines can be explored by typing

shtools.[tab]

where [tab] is the tab key. To read the documentation of a routine, such as PlmBar, enter

shtools.PlmBar?
